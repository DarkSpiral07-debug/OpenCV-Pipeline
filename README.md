# Autonomous Vision Pipeline

Real-time computer vision pipeline for autonomous vehicle perception using Python and OpenCV.

## Features

- Live camera capture
- Frame resizing
- Frame normalization
- Grayscale preprocessing
- Gaussian blur filtering
- Noise reduction
- Edge detection
- Corner detection
- ORB feature extraction
- FPS monitoring
- Screenshot capture

## Technologies Used

- Python
- OpenCV
- NumPy

## Future Scope

This pipeline can be extended for:

- Lane Keep Assist (LKA)
- Object Detection
- Object Tracking
- Autonomous Navigation

## Run

Install dependencies:

pip install -r requirements.txt

Run program:

python CameraPipeline.py

## Controls

- Press S to save screenshot
- Press Q to quit
